dot(b, out=None)¶

Dot product of two arrays.

Refer to numpy.dot for full documentation.

numpy.dot : equivalent function

>>> a = np.eye(2)
>>> b = np.ones((2, 2)) * 2
>>> a.dot(b)
array([[2.,  2.],
       [2.,  2.]])

This array method can be conveniently chained:

>>> a.dot(b).dot(b)
array([[8.,  8.],
       [8.,  8.]])
